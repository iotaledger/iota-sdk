You can generate an IOTA address using the `Client.get_addresses()` function that will return a list of tuples with the
generated addresses.

The following code example will:

1. Create a `Client` which will connect to the [Shimmer Testnet](https://api.testnet.shimmer.network).
2. Create a `SecretManager` from a [mnemonic](02_generate_mnemonic.mdx).
3. Generate a public address.
4. Generate an internal address.
5. Generate an offline address.
