The simplest block you can create is an empty block. You will only need connect your client to a node and then call the
`Client.block(&self)` function.

The following code example will:

1. Create a `Client` which will connect to the [Shimmer Testnet](https://api.testnet.shimmer.network).
2. Build and post an empty block.
