You can post a data block by adding hex encoded data as options when you [create a block](07_create_block.mdx).

The following code example will:

1. Create a `Client` which will connect to the [Shimmer Testnet](https://api.testnet.shimmer.network).
2. Create the `options` with the hex encoded data.
3. Create a `SecretManager` from a [mnemonic](02_generate_mnemonic.mdx).
4. [Create and post](07_create_block.mdx) the block with the data from step 2.
4. [Generate a public address](03_generate_addresses.mdx).
