# Run Code Examples

You can use the following how-to guides to get acquainted with IOTA SDK, from creating an account to minting NFTs and
native tokens.

You can start anywhere, but since the guides may build on previous examples if you are new to IOTA SDK
you should probably start by [creating a wallet account](./accounts_and_addresses/create_account.mdx) and then
carry on to the following guide.

## Requirements

### Hornet Node

Most of the examples require a [Hornet Node](https://wiki.iota.org/shimmer/hornet/welcome) running the
[Stardust](https://wiki.iota.org/shimmer/introduction/welcome) upgrade. You can connect to the
[Shimmer Testnet](#shimmer-testnet), a node you trust,
[run your own node](https://wiki.iota.org/shimmer/hornet/how_tos/using_docker) or
[run a local private tangle](https://wiki.iota.org/shimmer/hornet/how_tos/private_tangle).

#### Shimmer Testnet

You can use the following public, load-balanced endpoint to run code examples:

#### Node URL

```plaintext
https://api.testnet.shimmer.network
```

#### Faucet URL

```plaintext
https://faucet.testnet.shimmer.network/api/enqueue
```

## Run Code Examples


After you have [installed the IOTA SDK](#install-the-iota-sdk), you can browse the IOTA SDK code examples to get
acquainted with the `client` and `wallet` modules.
Please ensure you have followed the instructions to

<Tabs groupId="language">
<TabItem value="rust" label="Rust">

You can get a complete list of the available [Rust examples](https://github.com/iotaledger/iota-sdk/tree/develop/sdk/examples)
by running the following command:

```shell
cargo run --example
```

You can use the following command to run the examples, replacing `example_name` with any example name:

```rust
cargo run --example example_name --release
```

</TabItem>
<TabItem value="nodejs" label="Nodejs">

You can browse all the available [Node.js examples](https://github.com/iotaledger/iota-sdk/tree/develop/bindings/nodejs/examples)
in the repository. You can use the following command to run any example from the Node.js directory:

```bash
cd examples
yarn run-example ./[example_folder]/[example_file]
```

- Where `[example_file]` is the file name from the example folder, and `[example_folder]` is the folder name. For example:

```bash
yarn run-example  examples/client/00_get_info.ts
```

</TabItem>
<TabItem value="python" label="Python">

You can browse all the available [Python examples](https://github.com/iotaledger/iota-sdk/tree/develop/bindings/python/examples)
in the repository. You can use the following command to run any example from the Python bindings directory:

```bash
python3 example/[example file]
```

- Where `[example file]` is the file name from the example folder. For example:

```bash
python3 examples/client/00_get_info.py
```

</TabItem>
